# Cupón descuento carVertical

Discount codes for carVertical VIN decoder services in Spain. We use these for testing keywords like cupón descuento carVertical, código descuento carVertical, carVertical discount code, and carVertical coupon.

## Installation

Install `cupon-descuento-carvertical` via NPM:

```bash
npm install cupon-descuento-carvertical
```

## Usage

The package contains an array of objects representing discount codes.

In Node.js:

```javascript
var codes = require('cupon-descuento-carvertical')

console.log(codes)

// [
//   {
//     country: 'Spain',
//     website: 'https://consultarvehiculo.es/',

```
